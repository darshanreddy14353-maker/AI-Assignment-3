STATIC AND DYNAMIC ENVIRONMENT - README

Objective:
To simulate pathfinding in both static and dynamic environments.

Static Environment:
- Environment does not change
- Obstacles remain fixed
- Easier to compute path

Dynamic Environment:
- Obstacles may change during execution
- More realistic (like robots, self-driving cars)
- Requires adaptive algorithms

UGV (Unmanned Ground Vehicle):
- Robot that moves in grid
- Finds path from start to goal

Approach:
- Use grid representation
- Apply Dijkstra or similar algorithm
- Avoid obstacles while moving

Applications:
- Robotics
- Autonomous Vehicles
- Game AI

Conclusion:
Dynamic environments are complex but closer to real-world AI problems.
